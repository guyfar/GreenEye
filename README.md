## 功能特性

- **蓝光过滤** - 科学降低屏幕蓝光，减轻眼睛疲劳
- **护眼主题** - 4种预设主题（豆沙绿、羊皮纸、暖夜灯、深邃黑）
- **网站记忆** - 自动记住每个网站的偏好设置
- **实时预览** - 调节时即时看到效果

## 安装方法

### 从源码安装

1. 下载或克隆本仓库
2. 打开 Chrome/Dia 浏览器，访问 `chrome://extensions`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择项目文件夹

## 使用说明

1. 点击浏览器工具栏中的 GreenEye 图标
2. 开启「护眼模式」开关
3. 调节蓝光过滤强度（0-100%）
4. 选择喜欢的护眼主题
5. 点击「记住设置」为当前网站保存偏好

## 护眼主题

| 主题 | 背景色 | 适用场景 |
|------|--------|----------|
| 豆沙绿 | #C7EDCC | 日间长时阅读 |
| 羊皮纸 | #FAF9DE | 模拟纸质书籍 |
| 暖夜灯 | #FDF5E6 | 夜间柔和模式 |
| 深邃黑 | #1A1A1A | OLED 深色模式 |

## 技术栈

- Chrome Extension Manifest V3
- 原生 JavaScript（无框架依赖）
- SVG 颜色矩阵实现蓝光过滤

## 兼容性

- Google Chrome
- Microsoft Edge
- Dia Browser
- 其他 Chromium 内核浏览器
